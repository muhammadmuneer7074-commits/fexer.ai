Production AI SaaS — Chat, Voice, Image Generation, AI Projects.

| Layer | Tech |
|---|---|
| Frontend | Vanilla HTML + CSS + JS |
| Hosting | Netlify |
| Database + Auth | Supabase |
| Payments | Lemon Squeezy |
| AI | OpenAI (GPT-4o-mini, DALL-E-3, Whisper, TTS) |
| Search | Tavily |
| Automation | n8n on Railway |
